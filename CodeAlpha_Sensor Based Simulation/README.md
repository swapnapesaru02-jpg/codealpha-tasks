Automatic Light Control Using LDR Sensor

Project Overview

This project demonstrates an Automatic Light Control System using an LDR (Light Dependent Resistor) and Arduino Uno. The system automatically controls an LED based on surrounding light intensity.


---
 Objective

To design and simulate a system that:

Turns an LED ON in low light conditions

Turns OFF the LED in bright light conditions
using an LDR sensor.
 Components Used

Arduino Uno

LDR (Photoresistor)

LED

220Ω Resistor

10kΩ Resistor

Breadboard

Jumper Wires

 How It Works

The LDR changes its resistance based on light intensity. The Arduino reads this change through an analog pin.

When the light intensity is low (dark), the LDR resistance is high, and the LED turns ON.

When the light intensity is high (bright), the LDR resistance is low, and the LED turns OFF.


---

How to Use

1. Connect the components as per the circuit diagram.


2. Upload the code to Arduino Uno.


3. Power the circuit.


4. The LED will automatically turn ON/OFF based on surrounding light.
